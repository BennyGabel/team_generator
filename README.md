
# readme-generator

## Description
  Generates a README.md markup file. Using Node JS and inquirer

## Installation
1) Pre requisit, having Node js pre installed 
2) Download dependencies:     *"inquirer 8.2.4"*   &   *"Jest"*       - **npm i**
